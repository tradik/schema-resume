Overview

Schema Resume provides a self-hosted, comprehensive JSON-LD schema for representing professional resumes and CVs. It enables easy parsing, validation, and exchange of resume data between systems with semantic web integration, making it perfect for job boards, applicant tracking systems, and resume builders. No external dependencies required.

Quick Start

Basic Example

{ "@context": "https://tradik.github.io/schema-resume/schema.json", "$schema": "https://tradik.github.io/schema-resume/schema.json", "basics": { "name": "John Doe", "label": "Software Engineer", "email": "john.doe@example.com", "summary": "Experienced software engineer..." }, "work": [ { "name": "Tech Corp", "position": "Senior Software Engineer", "startDate": "2020-03", "highlights": [ "Reduced API response time by 40%" ] } ] }

Validation

Validate your resume JSON using AJV:

npm install -g ajv-cli ajv validate -s schema.json -d your-resume.json

Schema Sections

  • basics - Personal information, contact details, nationalities, and work authorization
  • work - Work experience and employment history
  • volunteer - Volunteer work and community involvement
  • education - Academic background and qualifications
  • awards - Professional awards and recognitions
  • certificates - Professional certifications
  • publications - Published works and articles
  • skills - Technical and professional skills with experience levels
  • tools - Specific software and platforms used professionally
  • languages - Language proficiencies
  • interests - Personal interests and hobbies
  • references - Professional references
  • projects - Personal and professional projects
  • meta - Schema metadata and versioning
